# 晨练 · 今日训练播放器

一个**零决策**的每日晨练小工具。打开就是今天该练的内容,点一下卡片直接内嵌播放 B 站跟练视频,像执行机器一样一节一节往下做。单文件、纯前端、无需安装。

## 特点

- 📅 **按星期自动排课**——每天固定「肩颈放松 + 八段锦」,力量项(腹 / 臀 / 手臂)三选一轮换,周日恢复。
- ▶️ **点卡片即播**——手风琴式展开,内嵌 B 站官方播放器,点标题条自动播放。
- 🔀 **一周任意切换**——顶部星期按钮,提前预览任何一天。
- ➕➖ **视频可增删**——每个环节维护自己的视频列表,粘贴 B 站链接/BV 号即可添加,不想要的一键删除。
- 💾 **本地存储**——你的视频列表、选择、打卡连续天数都存在浏览器 `localStorage`,不上传任何服务器。

## 默认课表

| 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 |
|------|------|------|------|------|------|------|
| 腹 | 臀 | 手臂 | 腹 | 臀 | 手臂 | 恢复 |

> 每天约 30 分钟:肩颈 5 分钟 + 八段锦约 12–15 分钟 + 当日力量项约 10 分钟。

## 使用

直接用浏览器打开 `morning-workout.html` 即可。

**内嵌播放建议**:B 站官方播放器要求宿主页面为 `http(s)` 协议。若直接双击 `file://` 打开出现黑屏,用本地小服务器打开更稳:

```bash
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000/morning-workout.html
```

每条视频下方都有「在 B 站打开 ↗」兜底链接,加载不出时可直接跳转观看。

手机上可在浏览器里「添加到主屏幕」,点开就像一个 App。

## 换成你自己的视频

打开某一节 → 底部粘贴框贴入 B 站视频链接或 BV 号 → 点「＋ 添加」。默认内置的视频可随时删除,换成全是你自己收藏的跟练。
